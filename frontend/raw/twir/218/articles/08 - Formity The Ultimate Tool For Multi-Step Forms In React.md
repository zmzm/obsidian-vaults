# Item 8: Formity: The Ultimate Tool For Multi-Step Forms In React

Source: [https://www.formity.app/blog/articles/formity-ultimate-tool-multi-step-forms-react](https://www.formity.app/blog/articles/formity-ultimate-tool-multi-step-forms-react)

Summary:
Formity is a new library focused on simplifying the creation of dynamic, multi-step forms in React. Unlike existing form libraries, Formity provides built-in support for advanced logic (conditions, loops), seamless integration with other form libraries, and advanced TypeScript type inference. The API is schema-driven, allowing developers to define form flow and behavior declaratively.

Key takeaways:
- Solves the pain point of building complex, multi-step forms with dynamic flow.
- Works alongside popular form libraries like React Hook Form and Formik.
- Schema-based approach enables advanced logic and strong type safety.
- Designed for scalability and developer ergonomics.
