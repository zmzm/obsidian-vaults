# Item 10: Bundle Size Investigation: A Step-by-Step Guide to Shrinking Your JavaScript

Source: [https://www.developerway.com/posts/bundle-size-investigation](https://www.developerway.com/posts/bundle-size-investigation)

Summary:
This guide walks through analyzing and reducing JavaScript bundle size in a React/Vite project using visualization tools. It demonstrates how large dependencies (like @mui/material and @mui/icons-material) can bloat bundles, and provides a systematic approach to identifying and eliminating unnecessary code. The article emphasizes the importance of regular bundle analysis and offers practical steps for investigation.

Key takeaways:
- Bundle analyzers (e.g., Rollup Plugin Visualizer) are essential for understanding and reducing bundle size.
- Large UI libraries and icon sets are common culprits for excessive bundle size.
- Systematic investigation involves identifying, understanding, and minimizing or replacing large dependencies.
- Regular analysis helps maintain fast, interactive React apps.

Recommendation: Read fully (read fully if you are actively working on bundle optimization)
