# Item 9: How to Build a Master-Detail UI with React Router

Source: [https://sergiodxa.com/tutorials/build-a-master-detail-ui-with-react-router](https://sergiodxa.com/tutorials/build-a-master-detail-ui-with-react-router)

Summary:
This tutorial demonstrates building a master-detail interface (like an inbox with detail view) using React Router, handling both in-app navigation and direct URL access. It covers route structure, context management, layout rendering, and redirect logic to ensure correct UI behavior in all navigation scenarios. The guide includes code examples and practical advice for robust, shareable master-detail UIs.

Key takeaways:
- Route configuration supports both browsing (master-detail) and direct access (detail-only) patterns.
- Context is used to determine layout state, enabling components to adapt rendering based on navigation context.
- Redirect logic ensures users see the appropriate view when accessing URLs directly or via navigation.
- The approach is applicable to dashboards, issue trackers, and similar apps requiring flexible navigation.
