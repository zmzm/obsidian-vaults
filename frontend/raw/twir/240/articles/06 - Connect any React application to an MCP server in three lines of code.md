# Item 6: Connect any React application to an MCP server in three lines of code

Source: [https://blog.cloudflare.com/connect-any-react-application-to-an-mcp-server-in-three-lines-of-code/](https://blog.cloudflare.com/connect-any-react-application-to-an-mcp-server-in-three-lines-of-code/)

Summary:
Cloudflare has open-sourced `use-mcp`, a React hook for connecting to Model Context Protocol (MCP) servers with minimal setup. The library abstracts transport, authentication (including OAuth 2.1), session management, and dynamic tool discovery, enabling rapid integration of AI-powered services. It also provides debugging, monitoring, and supports evolving MCP standards, with real-world usage by major companies.

Key takeaways:
- `use-mcp` connects React apps to MCP servers in just a few lines, handling all protocol and auth details.
- Supports real-time connection state, OAuth flows, and dynamic tool discovery.
- Includes debugging/logging tools and is future-proofed for evolving MCP specs.
- Open-sourced alongside an AI Playground demo for rapid prototyping and testing.
