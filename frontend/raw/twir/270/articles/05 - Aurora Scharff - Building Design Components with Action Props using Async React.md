# Item 5: Aurora Scharff - Building Design Components with Action Props using Async React

Source: [https://aurorascharff.no/posts/building-design-components-with-action-props-using-async-react/](https://aurorascharff.no/posts/building-design-components-with-action-props-using-async-react/)

Summary:
The article demonstrates practical patterns for building design components using Async React primitives like useTransition and useOptimistic. By encapsulating async actions and optimistic updates within components (via action props), the UX is improved and component consumers remain simple. Two examples—a TabList and an EditableText component—show how to handle pending states and optimistic UI updates internally.

Key takeaways:
- useTransition and useOptimistic enable responsive, optimistic UIs for async actions.
- The action prop pattern encapsulates async logic, reducing consumer complexity.
- Components manage their own pending and optimistic states, improving user feedback.
- Patterns are applicable to custom and future component libraries.
