# This Week in React #262 (MOC)

## TL;DR
- A critical (CVSS 10.0) server-side vulnerability in React.js (CVE-2025-55182) and Next.js (CVE-2025-66478) was disclosed, allowing potential remote code execution
- A major PR synchronizes the server and client implementations of React Flight, addressing deep cycle resolution and deferred error handling
- This post analyzes the confusion around the React2Shell vulnerability, explaining why many early PoCs and community theories were incorrect
- Cloudflare experienced a 25-minute outage affecting 28% of HTTP traffic, triggered by a rushed WAF configuration change in response to the React Server Components vulnerability
- Next.js published a security advisory for CVE-2025-66478, downstream of the React2Shell vulnerability
