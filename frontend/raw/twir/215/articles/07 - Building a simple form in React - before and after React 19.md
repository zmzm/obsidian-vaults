# Item 7: Building a simple form in React - before and after React 19

Source: [https://reactpractice.dev/articles/building-a-simple-form-in-react-before-and-after-react-19/](https://reactpractice.dev/articles/building-a-simple-form-in-react-before-and-after-react-19/)

Summary:
This article compares form handling in React before and after version 19, using a newsletter subscription example. React 19 introduces the useActionState hook and allows passing functions to the form action attribute, streamlining form state, validation, and submission. The new approach removes the need for controlled inputs and manual loading/error state management, simplifying form code.

Key takeaways:
- React 19’s useActionState and form action attribute reduce boilerplate and complexity in forms.
- Controlled inputs and manual state management are no longer required for many forms.
- Error and success states can be returned directly from the action handler.
- The new pattern aligns React forms more closely with web standards.
