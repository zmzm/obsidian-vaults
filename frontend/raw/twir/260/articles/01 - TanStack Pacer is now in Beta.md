# Item 1: TanStack Pacer is now in Beta

Source: [https://tanstack.com/pacer/latest](https://tanstack.com/pacer/latest)

Summary:
TanStack Pacer is a new, framework-agnostic utility library for debouncing, throttling, rate limiting, queuing, and batching operations. It offers both synchronous and asynchronous primitives, with a modular, type-safe API and dedicated React adapters and hooks. The library aims to optimize application performance and is currently in beta.

Key takeaways:
- Provides a comprehensive set of utilities for controlling function execution frequency and concurrency.
- Includes React-specific hooks (e.g., useDebouncer, useThrottler, useRateLimiter) for seamless integration.
- Designed to be framework-independent, but with ergonomic React adapters.
- Supports both synchronous and asynchronous workflows.
