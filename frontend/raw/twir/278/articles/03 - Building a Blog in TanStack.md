# Item 3: Building a Blog in TanStack

Source: [https://frontendmasters.com/blog/building-a-blog-in-tanstack-part-1-of-2/](https://frontendmasters.com/blog/building-a-blog-in-tanstack-part-1-of-2/)

Summary:
This article demonstrates building a blog using TanStack Start, focusing on server functions, routing, and static pre-rendering. Blog posts are written in Markdown, with metadata extracted using gray-matter, and dynamic discovery is handled via Vite's import.meta.glob. The post covers the setup for reading posts, extracting metadata, and wiring up routes and loaders, highlighting the isomorphic nature of TanStack loaders and the use of server functions for file system access. Part 2 will address static generation and deployment.

Key takeaways:
- TanStack Start enables modern SSR and routing, with server functions for backend logic.
- Markdown posts and metadata are dynamically discovered and parsed at build time.
- Isomorphic loaders require server functions to access file system data securely.
- The approach is practical for content-driven React apps needing SSR/static generation.
