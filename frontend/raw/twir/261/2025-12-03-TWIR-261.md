## TL;DR
- A critical unauthenticated remote code execution vulnerability (CVE-2025-55182, CVSS 10.0) was discovered in React Server Components, affecting several react-server-dom-* packages and major frameworks like Next.js, React Router, and others
- The React-Redux team proposes officially marking the connect API as deprecated in favor of useSelector and other modern hooks, as connect will not be updated for upcoming concurrent store APIs
- Francisco Miranda presents a project that reimagines the GitHub repository view with fully streamed, instant-loading UI, eliminating the familiar blue loading bar
