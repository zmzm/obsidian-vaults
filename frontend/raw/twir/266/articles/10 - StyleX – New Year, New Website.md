# Item 10: StyleX – New Year, New Website

Source: [https://stylexjs.com/blog/a-new-year-2026](https://stylexjs.com/blog/a-new-year-2026)

Summary:
StyleX announces a new website and playground, reflecting its growth and adoption in the React ecosystem. The new site is built with Waku (a minimal React framework supporting Server Components), Fumadocs for documentation, and showcases StyleX's latest APIs and improved integration. The playground now compiles StyleX in-browser via Babel, making it easier to try and share examples.

Key takeaways:
- StyleX is now widely adopted (Meta, Figma, Snowflake, HubSpot) and has new APIs for advanced styling.
- @stylexjs/unplugin simplifies project integration.
- New playground compiles StyleX in-browser for instant experimentation.
- Website rebuilt with Waku and Fumadocs, fully styled with StyleX.
