# Item 7: Kent C. Dodds - Migrating Remix v2 to React Router v7 with Cursor

Source: [https://kentcdodds.com/blog/how-i-used-cursor-to-migrate-frameworks](https://kentcdodds.com/blog/how-i-used-cursor-to-migrate-frameworks)

Summary:
Kent C. Dodds details his experience using the Cursor AI agent to upgrade a large project from Remix v2 to React Router v7, along with other major dependency upgrades. The process involved automated planning, handling of custom patches, and migration to new routing conventions, with the agent performing most tasks and Kent providing targeted feedback. The article highlights the benefits and limitations of AI-assisted migrations in real-world, complex codebases.

Key takeaways:
- AI agents like Cursor can automate much of the upgrade process, especially with good tests and documentation.
- Major framework migrations require careful handling of custom patches and routing conventions.
- Incremental adoption and feedback loops are key to successful, agent-assisted migrations.
- The approach is practical for large, mature codebases with existing test coverage.
