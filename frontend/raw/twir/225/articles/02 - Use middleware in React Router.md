# Item 2: Use middleware in React Router

Source: [https://sergiodxa.com/tutorials/use-middleware-in-react-router](https://sergiodxa.com/tutorials/use-middleware-in-react-router)

Summary:
This tutorial explains how to enable and use middleware in React Router starting with v7.3.0. It covers setup, key API changes, implementing custom middleware (like session management), and how to use middleware on both server and client sides. The article also addresses breaking changes to context handling and demonstrates how to type params for route-specific middleware.

Key takeaways:
- Middleware is enabled via the unstable_middleware flag in the router config.
- Context changes: AppLoadContext is replaced with unstable_RouterContextProvider, allowing middleware to share data.
- Middleware can be attached to routes and is available for both server and client loaders/actions.
- Route-specific typing is possible for middleware params.
