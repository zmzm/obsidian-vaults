# Item 6: What Do We Do with You, Old React?

Source: [https://www.manuelsanchezdev.com/blog/what-to-do-with-old-react-upgrading-to-modern-react](https://www.manuelsanchezdev.com/blog/what-to-do-with-old-react-upgrading-to-modern-react)

Summary:
This guide walks through modernizing legacy React projects (pre-16.8) by refactoring class components to hooks, updating Redux usage, and improving tests. It advocates for incremental upgrades rather than full rewrites, discusses when to consider alternative frameworks, and compares Redux Toolkit and lighter state libraries. Practical code examples illustrate the migration path from classes and legacy Redux to modern React patterns.

Key takeaways:
- Incremental migration from class components to hooks is feasible and recommended.
- Modern Redux Toolkit reduces boilerplate and improves maintainability.
- Consider lighter state libraries (Jotai, Zustand) for simpler apps.
- Upgrading tests to @testing-library/react is part of modernization.
