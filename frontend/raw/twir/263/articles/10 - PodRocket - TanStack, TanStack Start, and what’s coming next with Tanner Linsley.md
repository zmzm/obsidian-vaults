# Item 10: PodRocket - TanStack, TanStack Start, and what’s coming next with Tanner Linsley

Source: [https://podrocket.logrocket.com/tanstack-tanstack-start-and-whats-coming-next-with-tanner-linsley](https://podrocket.logrocket.com/tanstack-tanstack-start-and-whats-coming-next-with-tanner-linsley)

Summary:
This podcast features Tanner Linsley discussing the evolution of TanStack projects, including React Query, React Table, and the new TanStack Start framework. The conversation covers TanStack’s headless philosophy, performance optimizations, client-first SSR, routing, and data loading, as well as the challenges of forms, authentication, and sustainable open source development.

Key takeaways:
- TanStack Start introduces a client-first approach to SSR, routing, and data loading.
- Emphasis on composability, type safety, and gradual adoption across TanStack projects.
- Discussion of tradeoffs with React Server Components and other meta-frameworks.
- Insights into open source sustainability and TanStack’s future roadmap.
