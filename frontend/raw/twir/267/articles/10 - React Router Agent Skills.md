# Item 10: React Router Agent Skills

Source: [https://github.com/remix-run/agent-skills](https://github.com/remix-run/agent-skills)

Summary:
The React Router team introduces "Agent Skills," structured documentation modules that teach AI coding agents how to use React Router correctly in various modes (framework, data, declarative). Skills are installed into projects and provide up-to-date patterns, references, and code samples for AI agents to generate accurate code.

Key takeaways:
- Agent Skills deliver current, mode-specific React Router usage patterns for AI agents and developers.
- Covers framework, data, and declarative routing modes with quick references and detailed docs.
- Helps prevent outdated or incorrect code generation by AI tools.
- Easy installation via npx skills add.
