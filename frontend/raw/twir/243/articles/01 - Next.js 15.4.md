# Item 1: Next.js 15.4

Source: [https://nextjs.org/blog/next-15-4](https://nextjs.org/blog/next-15-4)

Summary:
Next.js 15.4 delivers significant performance and stability improvements, especially around Turbopack, which now passes all integration tests for production builds and is powering vercel.com. The release also previews features coming in Next.js 16, such as unified caching, optimized client-side routing, improved DevTools, and stable Node.js middleware. Developers are encouraged to try Turbopack again and experiment with upcoming features via the canary channel. Numerous bug fixes, performance tweaks, and new features are included in this release.

Key takeaways:
- Turbopack integration is now robust, passing all 8298 integration tests and powering high-traffic sites.
- Next.js 16 will consolidate caching features, improve client-side routing, and enhance DevTools/debugging.
- Node.js middleware is stable; deployment adapters and deprecations (Node 18, AMP) are coming.
- Many notable improvements and fixes (e.g., error handling, cache, router prefetching, RSC query preservation).
