# Item 7: DexieJs: Reactive local state in React

Source: [https://www.typeonce.dev/article/dexie-js-reactive-local-state-in-react](https://www.typeonce.dev/article/dexie-js-reactive-local-state-in-react)

Summary:
This article demonstrates how to use Dexie.js and its React hooks to manage reactive local state in React applications, leveraging IndexedDB for persistence. It introduces a custom useDexieQuery hook for live queries and a useCustomAction hook (with React 19’s useActionState and startTransition) for async writes. The approach eliminates the need for global stores or manual loading state management, offering a streamlined developer experience for local-first apps.

Key takeaways:
- Dexie.js + dexie-react-hooks enable reactive, persistent local state.
- useDexieQuery wraps live queries with error/loading handling.
- useCustomAction (with useActionState) simplifies async actions and state updates.
- No need for global state libraries; components always see up-to-date data.
