# Item 1: Storybook 9

Source: [https://storybook.js.org/blog/storybook-9/](https://storybook.js.org/blog/storybook-9/)

Summary:
Storybook 9 introduces major improvements in component testing, integrating with modern tools like Vitest and Playwright to provide fast, comprehensive interaction, accessibility, and visual tests. The release also brings a leaner core, automatic story generation, tag-based organization, enhanced story globals, and significant framework support upgrades, including Vite-powered Next.js and Svelte 5. These features aim to make UI state coverage, debugging, and test maintenance easier and more scalable for large projects.

Key takeaways:
- Storybook Test enables running all interaction, accessibility, and visual tests across all stories, with detailed coverage and debugging tools.
- The new version is 48% smaller, with optional docs and test packages, and offers automatic story and test generation.
- Tag-based organization and story globals improve story management and contextual testing.
- Framework support is expanded, notably with Vite-powered Next.js and Svelte 5 compatibility.
