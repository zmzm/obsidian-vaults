# Item 1: Wake up Remix - The plan for Remix v3

Source: [https://remix.run/blog/wake-up-remix](https://remix.run/blog/wake-up-remix)

Summary:
Remix v3 is being reimagined as a modular, web-first toolkit, breaking away from its previous tight coupling with React Router and even React itself. The new direction focuses on simplicity, performance, and direct use of web APIs, with minimal dependencies and a composable architecture. Remix 3 will start with a fork of Preact and aims to provide a lighter, more web-aligned development experience. The team is emphasizing principles like model-first development, runtime-centric design, and cohesive distribution of packages.

Key takeaways:
- Remix v2 merged much of its core into React Router v7, which now supports RSC and is widely adopted.
- Remix 3 will not depend on React, instead starting with Preact and building a new component model.
- The focus is on modularity, minimal dependencies, and leveraging native Web APIs.
- The project is guided by principles such as model-first development, runtime focus, and composability.
