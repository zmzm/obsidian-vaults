# Item 9: Building the Twitter 1.0 Web Experience

Source: [https://largeapps.dev/case-studies/twitter/](https://largeapps.dev/case-studies/twitter/)

Summary:
This case study chronicles the development of Twitter Lite, a Progressive Web App (PWA) built with React to deliver a fast, accessible, and universal web experience across devices. It covers the business motivations, technical strategies, and lessons learned from 2015–2022, including performance optimizations, PWA best practices, and the impact of a single codebase for all platforms. The article emphasizes how Twitter Lite’s architecture enabled rapid growth, improved engagement, and broad accessibility.

Key takeaways:
- Twitter Lite’s PWA leveraged React to provide a performant, installable, and data-efficient experience for users worldwide.
- A universal codebase enabled consistent updates and features across mobile, tablet, and desktop platforms.
- Performance and accessibility improvements led to significant increases in engagement and reduced bounce rates.
- The case study offers practical insights for building large-scale, cross-platform web applications.
