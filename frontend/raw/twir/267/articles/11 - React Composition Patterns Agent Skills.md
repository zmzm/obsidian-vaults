# Item 11: React Composition Patterns Agent Skills

Source: [https://skills.sh/vercel-labs/agent-skills/vercel-composition-patterns](https://skills.sh/vercel-labs/agent-skills/vercel-composition-patterns)

Summary:
Vercel Labs offers a set of "Agent Skills" for React composition patterns, aimed at both human and AI developers. The guide covers architectural, state management, and implementation patterns, with a focus on React 19 APIs and best practices for scalable, maintainable components. Each rule includes explanations and code examples, supporting refactoring and library development.

Key takeaways:
- Provides actionable rules for component architecture, state management, and React 19-specific APIs.
- Emphasizes composition over boolean props, explicit variants, and context/provider patterns.
- Includes React 19 guidance (e.g., using use() instead of useContext, avoiding forwardRef).
- Designed for both manual and AI-assisted code refactoring.

Recommendation: Read fully (read fully if refactoring large component libraries or adopting React 19 patterns)
