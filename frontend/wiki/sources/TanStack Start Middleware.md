# TanStack Start Middleware

This source captures middleware as one of the strongest explicit framework primitives in TanStack Start.

## Summary

- Middleware can wrap server functions on both client and server sides.
- It supports cross-cutting concerns such as observability, tracing, and shared context propagation.
- The model is strongly typed and explicit about where request-handling logic is inserted.

## Why This Source Matters

- It gives the `TanStack Start` page a concrete architectural capability rather than only comparison rhetoric.
- It helps explain why some teams see TanStack Start as clearer and more composable than framework models with more implicit behavior.

## Caveats

- The source is strongest for teams already thinking in explicit full-stack React primitives.
