# This Week in React #237 (MOC)

## TL;DR
- Storybook 9 introduces major improvements in component testing, integrating with modern tools like Vitest and Playwright to provide fast, comprehensive interaction, accessibility, and visual tests
- This guide details the process for migrating a Next.js App Router project to TanStack Start, outlining prerequisites, dependency changes, and configuration updates
- A low-severity vulnerability affecting Next.js dev server (versions 13.0.0–14.2.29 and 15.0.0–15.2.1) allowed cross-site WebSocket hijacking and cross-origin script inclusion due to missing origin validation
- Dan Abramov explores the concept of "progressive JSON," applying ideas from progressive JPEGs to JSON data transfer
- This article examines the evolution from server-rendered HTML (single roundtrip for navigation and data) to client-driven JSON APIs (multiple fetches per navigation), highlighting inefficiencies and the tension between encapsulation and performance
