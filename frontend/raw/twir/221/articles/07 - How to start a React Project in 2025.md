# Item 10: How to start a React Project in 2025

Source: [https://www.robinwieruch.de/react-starter/](https://www.robinwieruch.de/react-starter/)

Summary:
This annual overview compares modern ways to start a React project, focusing on Vite (as a CRA replacement), Next.js (for full-stack and SSR), and Astro (for content-focused sites). It discusses the pros and cons of each, including configuration, performance, learning curve, and ecosystem support. The guide helps developers choose the right starter based on project needs and desired flexibility.

Key takeaways:
- Vite is the recommended lightweight, framework-agnostic CRA successor.
- Next.js offers full-stack capabilities and advanced SSR/SSG features.
- Astro is optimal for content-heavy, SEO-focused sites.
- Choice depends on project requirements and team expertise.
