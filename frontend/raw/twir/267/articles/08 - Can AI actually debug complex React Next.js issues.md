# Item 8: Can AI actually debug complex React/Next.js issues?

Source: [https://www.developerway.com/posts/debugging-with-ai](https://www.developerway.com/posts/debugging-with-ai)

Summary:
The author tests whether LLMs can debug and fix real-world React/Next.js issues, comparing AI-generated fixes to manual investigation. In the case study, the LLM successfully patches a broken user profile page by adding missing fields, but the author validates the root cause and explains the debugging process in detail. The article highlights both the promise and limitations of AI debugging, emphasizing the importance of understanding underlying schemas and data flow.

Key takeaways:
- LLMs can sometimes fix surface-level bugs in React/Next.js apps, but may not always identify or address root causes.
- Manual debugging remains essential for understanding schema validation errors and complex data flows.
- The article provides a practical walkthrough of debugging with Zod, TanStack Query, and Next.js.
- Encourages developers to use AI as an assistant, not a replacement for expertise.
