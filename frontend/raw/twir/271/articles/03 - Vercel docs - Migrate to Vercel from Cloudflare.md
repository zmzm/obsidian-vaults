# Item 3: Vercel docs - Migrate to Vercel from Cloudflare

Source: [https://vercel.com/kb/guide/migrate-to-vercel-from-cloudflare](https://vercel.com/kb/guide/migrate-to-vercel-from-cloudflare)

Summary:
This migration guide details how to move applications from Cloudflare Pages/Workers to Vercel, covering project setup, DNS, build configuration, environment variables, and feature mapping. It explains the differences in deployment, environment management, and how to translate Cloudflare-specific features to Vercel equivalents. The guide is practical for teams planning a migration or evaluating platform differences.

Key takeaways:
- Vercel offers auto-detection for popular frameworks and streamlined CI/CD workflows.
- DNS migration can be performed with zero downtime if following the documented steps.
- Environment variables and secrets must be manually transferred and scoped appropriately.
- Vercel provides open-source and open-standard alternatives to Cloudflare's proprietary features.

Recommendation: Read fully (read fully if planning a migration)
