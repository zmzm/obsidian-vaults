# This Week in React #234 (MOC)

## TL;DR
- TanStack DB is a new reactive client-side data store currently in beta, designed to optimize data loading and client performance for modern apps
- This PR introduces the react-server-dom-vite package, aiming to enable React Server Components (RSC) support in Vite-based projects
- This RFC proposes unifying the imperative query methods in React Query, which currently include fetchQuery, prefetchQuery, and ensureQueryData (plus their infinite variants)
- This post explains how React Server Components (RSC) can be used to generate static sites by running server logic at build time, blurring the lines between "server" and "static" frameworks
- This article demonstrates how to create factory functions for generating custom React hooks, reducing repetitive code when similar hooks are needed across a codebase
