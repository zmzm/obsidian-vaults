# Item 1: Parcel v2.14.0

Source: [https://parceljs.org/blog/v2-14-0/](https://parceljs.org/blog/v2-14-0/)

Summary:
Parcel v2.14.0 introduces beta support for React Server Components (RSC), enabling integration into existing apps for server, client, or static rendering. The release also adds first-class MDX support, a CLI for scaffolding new apps, a new React error overlay, and improved browser caching via HTML import maps. Parcel’s approach allows incremental adoption of RSC without a full framework rewrite, supporting both static and dynamic rendering scenarios. The update is significant for developers seeking flexible, modern React build tooling.

Key takeaways:
- Beta support for React Server Components with easy incremental adoption in existing apps.
- Out-of-the-box static site generation and MDX support, including automatic TOC and metadata extraction.
- New CLI (create-parcel) for rapid project scaffolding and a React error overlay for better DX.
- Unified build process for server/client code, supporting code splitting and environment boundaries.
