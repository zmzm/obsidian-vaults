# Item 12: Open-sourcing our microfrontend React bridge

Source: [https://alexocallaghan.com/open-sourcing-mfe-react-bridge](https://alexocallaghan.com/open-sourcing-mfe-react-bridge)

Summary:
Mintel has open-sourced @mintel/mfe-react-bridge, a small library that enables independent React version upgrades across multiple micro-frontends. The package provides helper functions and TypeScript types to render micro-frontends using different React versions, addressing issues found with @module-federation/bridge-react.

Key takeaways:
- Enables independent React upgrades in microfrontend architectures.
- Supports React 17 and 18+, with TypeScript types and tests.
- Solves re-rendering and over-complexity issues of existing solutions.
- Open source and available on npm for broader use.

Recommendation: Read fully (read fully if you manage React microfrontends or versioning)
