# This Week in React #259 (MOC)

## TL;DR
- PostHog shares lessons learned from pricing their suite of developer-focused products
- A new documentation PR clarifies how to correctly create and use Promises with React’s experimental use() hook for Suspense-enabled data fetching
- Next.js introduces a get_routes tool that scans the filesystem to discover all routes in a project, covering both App Router and Pages Router entry points
- A new next experimental-analyze CLI command generates bundle analyzer data for Next.js projects without writing chunks to disk, and can optionally serve the analyzer UI locally
- The next upgrade command is a convenience alias for running @next/codemod@canary upgrade, using the current package manager
