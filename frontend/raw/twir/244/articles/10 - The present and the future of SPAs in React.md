# Item 10: The present and the future of SPAs in React

Source: [https://www.felgus.dev/blog/future-spa](https://www.felgus.dev/blog/future-spa)

Summary:
This article explores the evolution and future of SPAs in React, focusing on patterns like render-as-you-fetch, concurrent features, and preloading. It discusses how modern routers (React Router v7, TanStack Router) and React features (Suspense, useTransition, Activity, ViewTransition) improve UX and performance. The author advocates for adopting these patterns to build faster, more interactive SPAs, even as server components gain popularity.

Key takeaways:
- Render-as-you-fetch pattern reduces waterfalls and improves perceived performance.
- Concurrent features (useTransition, useDeferred) benefit all apps, not just large ones.
- Preloading via router Link components and new React APIs (Activity, ViewTransition) can further optimize UX.
- SPAs remain relevant and can leverage modern React innovations.
