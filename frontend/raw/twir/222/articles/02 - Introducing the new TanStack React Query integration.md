# Item 5: Introducing the new TanStack React Query integration

Source: [https://sdk.vercel.ai/cookbook/next/human-in-the-loop](https://sdk.vercel.ai/cookbook/next/human-in-the-loop)

Summary:
tRPC has released a new integration with TanStack React Query, making the API more native and familiar to TanStack Query users. The new client uses standard QueryOptions and MutationOptions interfaces, removing an abstraction layer and improving compatibility with React's rules of hooks. Migration is encouraged for new and existing projects, with both classic and new clients able to coexist.

Key takeaways:
- New integration is simpler, more idiomatic, and leverages TanStack Query's native interfaces.
- Reduces learning curve and documentation overhead.
- Classic client will be maintained but not enhanced; migration is recommended.
- Codemod and migration docs are available.
