# Item 9: TanStack Start: New competitor to Next.js

Source: [https://ondrejvelisek.github.io/tanstack-start-new-competitor-to-nextjs/](https://ondrejvelisek.github.io/tanstack-start-new-competitor-to-nextjs/)

Summary:
TanStack Start is compared to Next.js, focusing on routing, typesafety, data fetching, and performance. Start offers flat-route definitions, end-to-end typesafe routing (including query params), and flexible file conventions. The article highlights how competition has driven Next.js to improve its own typesafety features, but Start remains ahead in some areas like query param typing and route inference.

Key takeaways:
- TanStack Start provides more robust typesafe routing and query param validation (e.g., via Zod) than Next.js.
- Flat-route definitions and file naming conventions offer flexibility and readability.
- Next.js has improved with typedRoutes, but still lags in dynamic route inference and query param typing.
- Both frameworks support SSR, streaming, and similar advanced features.
