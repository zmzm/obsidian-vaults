# Item 1: React Labs: View Transitions, Activity, and more

Source: [https://react.dev/blog/2025/04/23/react-labs-view-transitions-activity-and-more](https://react.dev/blog/2025/04/23/react-labs-view-transitions-activity-and-more)

Summary:
This React Labs update introduces two new experimental features—View Transitions and Activity—now available for testing in the latest experimental React releases. View Transitions enable declarative UI animations for transitions like navigation, leveraging the browser’s View Transition API, while Activity provides a way to show or hide parts of the UI. The post also shares progress on upcoming features such as React Performance Tracks, a Compiler IDE Extension, Automatic Effect Dependencies, Fragment Refs, and Concurrent Stores. The documentation for these features is newly published, and developers are encouraged to try them and provide feedback.

Key takeaways:
- View Transitions allow developers to declaratively animate UI changes (e.g., navigation, expanding sections) using a new <ViewTransition> component, integrating with browser APIs for smooth cross-fade and customizable animations.
- Activity, now shipped in react@19.2, is a component for conditionally showing or hiding UI segments.
- The experimental features are stable for production testing, but APIs may change based on user feedback.
- Additional features in development include performance tracking, IDE tooling, improved effect dependency management, and advanced state management patterns.
