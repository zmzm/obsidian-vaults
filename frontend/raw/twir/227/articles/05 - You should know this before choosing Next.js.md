# Item 5: You should know this before choosing Next.js

Source: [https://eduardoboucas.com/posts/2025-03-25-you-should-know-this-before-choosing-nextjs/](https://eduardoboucas.com/posts/2025-03-25-you-should-know-this-before-choosing-nextjs/)

Summary:
Written by a Netlify engineer, this opinion piece critiques the openness and governance of Next.js, particularly in light of the recent middleware security incident. It highlights the lack of official adapter support, proprietary build outputs, and challenges faced by non-Vercel providers in supporting the full Next.js feature set. The article calls for clearer boundaries between the open-source project and Vercel’s commercial interests, and greater transparency for the community.

Key takeaways:
- Next.js’s lack of adapters and proprietary build output complicate deployment on non-Vercel platforms.
- The recent security incident exposed communication and governance shortcomings.
- OpenNext and other providers are pushing for more open, portable deployment options.
- The ecosystem is awaiting promised improvements, but progress has been slow.
