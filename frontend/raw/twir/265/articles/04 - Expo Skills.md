# Item 4: Expo Skills

Source: [https://www.callstack.com/blog/announcing-react-native-best-practices-for-ai-agents](https://www.callstack.com/blog/announcing-react-native-best-practices-for-ai-agents)

Summary:
Expo Skills is an official set of AI agent skills from the Expo team, designed to assist with building, deploying, and debugging Expo apps. These skills are optimized for use with Claude Code but are compatible with any AI agent. Installation instructions are provided for both Claude Code and Cursor, and the skills are auto-discovered based on context when developers ask Expo-related questions.

Key takeaways:
- Official Expo skills streamline common tasks like UI building, data fetching, and deployment.
- Skills are context-aware and auto-invoked by agents, requiring minimal manual setup.
- Installation is supported for multiple agent platforms (Claude Code, Cursor, bunx).
- Skills are maintained under an MIT license and are regularly updated.
