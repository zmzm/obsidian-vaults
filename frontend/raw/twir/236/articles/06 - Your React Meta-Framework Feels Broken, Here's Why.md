# Item 6: Your React Meta-Framework Feels Broken, Here's Why

Source: [https://rwsdk.com/blog/your-react-meta-framework-feels-broken](https://rwsdk.com/blog/your-react-meta-framework-feels-broken)

Summary:
This opinion piece critiques the complexity and abstraction of modern React meta-frameworks, arguing that excessive indirection and convention obscure the underlying platform. It contrasts frameworks like Next.js and Remix with RedwoodSDK, which aims to minimize magic and work directly with web standards, offering explicit routing and server logic in plain TypeScript.

Key takeaways:
- Many frameworks introduce hidden conventions and magic exports, making code harder to reason about.
- RedwoodSDK promotes explicit, platform-aligned APIs with minimal abstraction.
- The goal is to let developers use JavaScript and web APIs directly, reducing cognitive overhead.
- Example comparisons show how RedwoodSDK avoids framework-specific magic.
