# This Week in React #263 (MOC)

## TL;DR
- The React team disclosed new high- and medium-severity vulnerabilities in React Server Components, including denial of service (DoS) and source code exposure issues
- Next.js issued a security advisory tracking the impact of the React Server Components vulnerabilities (DoS and source code exposure) on Next.js applications using the App Router
- This podcast episode discusses the React2Shell vulnerability, focusing on how server-side features in React 19 expanded the attack surface and the broader implications for JavaScript supply chain security
- Storybook disclosed a vulnerability where environment variables from .env files could be bundled into published Storybooks, potentially exposing secrets if the build directory contained a .env file and certain code patterns were used
- The React Aria documentation provides a comprehensive guide to installing, composing, and styling accessible UI components
