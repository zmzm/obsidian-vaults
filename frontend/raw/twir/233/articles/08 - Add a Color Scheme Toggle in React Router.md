# Item 9: Add a Color Scheme Toggle in React Router

Source: [https://sergiodxa.com/tutorials/add-a-color-scheme-toggle-in-react-router](https://sergiodxa.com/tutorials/add-a-color-scheme-toggle-in-react-router)

Summary:
This tutorial demonstrates how to implement a user-selectable color scheme (light, dark, system) toggle in a React Router app, using cookies for persistence and Tailwind for styling. It covers storing preferences securely, reading them on the server to avoid UI flashes, and integrating with app loaders and actions. The approach ensures a seamless, flicker-free theme experience for users, with code samples provided.

Key takeaways:
- Use cookies (not localStorage) to persist color scheme and avoid UI flashes on reload.
- Integrate color scheme logic into server loaders and React Router’s data APIs.
- Tailwind can be configured to support both class-based and system-based color schemes.
- Provides robust, production-ready code examples for theme toggling.
