Summary:
React Email 6.0 introduces a new open-source visual editor, a unified package structure, and new templates for common use cases. The editor is highly extensible, allowing developers to build custom blocks and themes, and outputs email-ready HTML compatible with major inbox providers. The release also simplifies package management and provides clear upgrade instructions from previous versions. React Email has seen significant growth, with 2M weekly downloads and nearly 200 contributors.

- Unified package structure simplifies imports and dependency management.
- New professionally designed templates for authentication and e-commerce are included.
