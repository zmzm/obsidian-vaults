# Item 5: React Server Components support without a framework

Source: [https://krasimirtsonev.com/blog/article/vanilla-react-server-components-with-no-framework](https://krasimirtsonev.com/blog/article/vanilla-react-server-components-with-no-framework)

Summary:
Forket is a new tool enabling React Server Components (RSC) without relying on frameworks like Next.js. It works by splitting code into client and server bundles, managing boundaries, and handling serialization and hydration manually. The article details the technical approach, challenges, and how Forket enables RSC patterns in custom setups, providing flexibility for large or legacy codebases.

Key takeaways:
- Forket allows RSC usage outside of established frameworks, increasing flexibility.
- Handles code splitting, serialization, and hydration for server/client boundaries.
- Useful for teams unable or unwilling to migrate to Next.js but wanting RSC features.
- Technical deep dive with practical examples for advanced React developers.
