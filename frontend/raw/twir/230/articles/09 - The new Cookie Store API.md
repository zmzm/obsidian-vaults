# Item 9: The new Cookie Store API

Source: [https://fotis.xyz/posts/the-new-cookie-store-api/](https://fotis.xyz/posts/the-new-cookie-store-api/)

Summary:
This article introduces the Cookie Store API, a modern, asynchronous browser API for structured cookie management, now supported in Chromium and Safari. It replaces the string-based, synchronous `document.cookie` with a more ergonomic and performant interface, including change events for real-time state updates. The post provides usage examples and discusses benefits for state management and bundle size.

Key takeaways:
- Cookie Store API enables async, structured cookie access and manipulation.
- Change events allow for reactive state management tied to cookies.
- Reduces need for third-party libraries and improves performance, especially with large cookies.
- Supported in Chromium and Safari (as of March 2025), with Firefox support pending.
