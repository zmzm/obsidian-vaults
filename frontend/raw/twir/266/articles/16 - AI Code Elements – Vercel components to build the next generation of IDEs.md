# Item 16: AI Code Elements – Vercel components to build the next generation of IDEs

Source: [https://vercel.com/changelog/ai-code-elements](https://vercel.com/changelog/ai-code-elements)

Summary:
Vercel has released a suite of composable React components (AI Code Elements) for building modern IDEs, coding apps, and background agents. Components include agent configuration, code blocks, commit history, environment variables, file trees, package info, sandboxes, schema displays, snippets, stack traces, terminals, and test results. The components are installable via npx and designed for extensibility and integration.

Key takeaways:
- Provides a comprehensive set of React components for building developer tools and IDEs.
- Supports advanced features: code blocks, file trees, test results, terminals, and more.
- Components are composable, performant, and easy to integrate.
- Designed to accelerate development of AI-powered coding environments.
