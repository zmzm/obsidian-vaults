# Item 14: PodRocket - What's new in React 19.2 with Shruti Kapoor

Source: [https://podrocket.logrocket.com/react-19-1-shruti-kapoor](https://podrocket.logrocket.com/react-19-1-shruti-kapoor)

Summary:
This podcast episode covers new features in React 19.2, including the Activity component, View Transitions, useEffectEvent, server component improvements, partial pre-rendering, Suspense boundary batching, the stable React Compiler, and enhanced Chrome DevTools performance tracks. It also discusses Next.js 16 support and updates to the ESLint react-hooks plugin.

Key takeaways:
- React 19.2 introduces significant performance and developer experience improvements.
- New features include Activity, View Transitions, and advanced server component capabilities.
- The React Compiler is now stable, enabling auto-memoization.
- Chrome DevTools integration and Next.js 16 support are highlighted.
