# Item 15: React Trends in 2025

Source: [https://www.robinwieruch.de/react-trends/](https://www.robinwieruch.de/react-trends/)

Summary:
This annual trends article highlights the mainstream adoption of React Server Components (RSC), the evolution of React Server Actions and Server Functions, and significant improvements to React Forms in React 19. It discusses the increasing importance of framework choice, the growing complexity of the React ecosystem, and the shift toward server-driven features.

Key takeaways:
- RSC is becoming a standard primitive, with broader framework support expected in 2025.
- React Server Functions (RSF) unify data fetching and mutations for both server and client components.
- React 19 introduces new form handling capabilities and hooks.
- Choosing the right framework is increasingly critical as the ecosystem diversifies.
