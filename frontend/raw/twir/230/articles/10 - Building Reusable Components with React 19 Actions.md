# Item 10: Building Reusable Components with React 19 Actions

Source: [https://aurorascharff.no/posts/building-reusable-components-with-react19-actions/](https://aurorascharff.no/posts/building-reusable-components-with-react19-actions/)

Summary:
This post explains how React 19 Actions, combined with hooks like useTransition and useOptimistic, can simplify handling pending states, errors, and optimistic UI updates in reusable components. The example focuses on a RouterSelect component that updates URL params and provides immediate user feedback during navigation transitions. The approach leverages React 19's new APIs to streamline component logic and improve UX.

Key takeaways:
- React 19 Actions and useTransition enable clean handling of async state and transitions.
- useOptimistic provides instant UI feedback for pending updates.
- Exposing an action property allows parent components to inject custom logic.
- The pattern supports building complex, reusable components with better user experience.
