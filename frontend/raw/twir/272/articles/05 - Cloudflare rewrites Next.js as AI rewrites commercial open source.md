# Item 5: Cloudflare rewrites Next.js as AI rewrites commercial open source

Source: [https://newsletter.pragmaticengineer.com/p/the-pulse-cloudflare-rewrites-nextjs](https://newsletter.pragmaticengineer.com/p/the-pulse-cloudflare-rewrites-nextjs)

Summary:
Cloudflare used AI agents to rewrite most of Next.js in a week, producing 'vinext'—a Vite-based, drop-in replacement for Next.js optimized for Cloudflare Workers. This event highlights how AI can rapidly disrupt commercial open source models by replicating complex frameworks at low cost. The article discusses the implications for proprietary build outputs, the role of test suites, and how AI may shift the defensibility of open source projects from code to community and infrastructure.

Key takeaways:
- AI can now replicate complex frameworks quickly and cheaply, challenging existing business models.
- Cloudflare's vinext replaces Turbopack with Vite, standardizing build outputs for broader deployment.
- Proprietary build formats and private test suites may become new defensive strategies.
- The value of open source may shift toward support, community, and infrastructure as code becomes easier to replicate.
