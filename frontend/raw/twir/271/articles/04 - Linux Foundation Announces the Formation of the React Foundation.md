# Item 4: Linux Foundation Announces the Formation of the React Foundation

Source: [https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-react-foundation](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-react-foundation)

Summary:
The Linux Foundation has launched the React Foundation, with Meta contributing the React project and major industry players joining as founding members. This move ensures React and React Native are governed openly and sustainably, with the aim of fostering community-driven innovation. The announcement includes endorsements from Amazon, Microsoft, Vercel, and others, emphasizing React's importance as digital infrastructure.

Key takeaways:
- React and React Native are now under open, neutral governance via the Linux Foundation.
- Major tech companies are committed as founding members, ensuring broad ecosystem support.
- The move is intended to secure React's long-term sustainability and community-driven evolution.
- Meta remains involved but is no longer the sole steward.
