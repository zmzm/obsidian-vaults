# Item 7: Beyond React.memo: Smarter Ways to Optimize Performance

Source: [https://cekrem.github.io/posts/beyond-react-memo-smarter-performance-optimization/](https://cekrem.github.io/posts/beyond-react-memo-smarter-performance-optimization/)

Summary:
This article argues that React.memo is often overused for performance optimization and suggests composition-based patterns as simpler, more maintainable alternatives. It explains React’s rendering model, demonstrates how moving state down and using children as props can reduce unnecessary re-renders, and provides practical code examples.

Key takeaways:
- React.memo adds complexity and is not always the best solution for avoiding re-renders.
- Moving state closer to where it’s used (component composition) isolates updates and improves performance.
- Using children as props leverages React’s rendering model for efficient updates.
- Understanding React’s re-render behavior is key to effective optimization.
