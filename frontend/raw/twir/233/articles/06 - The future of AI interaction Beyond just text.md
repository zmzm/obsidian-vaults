# Item 7: The future of AI interaction: Beyond just text

Source: [https://www.epicai.pro/the-future-of-ai-interaction-beyond-just-text-w22ps](https://www.epicai.pro/the-future-of-ai-interaction-beyond-just-text-w22ps)

Summary:
The article discusses the Model Context Protocol (MCP) and its potential to enable richer AI-driven user interfaces, moving beyond text-based interactions. Drawing on demos from ReactConf 2024, it argues that users want AI assistants to return interactive UIs (e.g., maps, stopwatches, data visualizations) rather than just text responses. The author predicts that UI development will remain essential, with AI augmenting but not replacing the need for interactive, visual experiences.

Key takeaways:
- MCP enables AI applications to serve dynamic, interactive UIs, not just text.
- Users prefer actionable, visual interfaces for tasks like navigation, timing, and data exploration.
- UI development skills will remain in demand, as AI-generated UIs need to be usable and branded.
- React Server Components may play a role in future AI-driven UI architectures.
