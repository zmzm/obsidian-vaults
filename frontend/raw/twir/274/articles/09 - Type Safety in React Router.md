# Item 9: Type Safety in React Router

Source: [https://raphaelbronsveld.com/blog/type-safety-in-react-router](https://raphaelbronsveld.com/blog/type-safety-in-react-router)

Summary:
React Router has improved type safety with generated types for routes, loader data, and actions, plus new hooks like `useRoute` and utilities like `href()`. These features reduce runtime errors and make navigation and data access more reliable and maintainable.

Key takeaways:
- Generated types provide type-safe access to route params, loader data, and action data.
- `useRoute` hook gives type-safe access to route data by route ID.
- `href()` utility ensures navigation links are type-checked and updated with route changes.
- Type safety features are available in framework mode, not library mode.
