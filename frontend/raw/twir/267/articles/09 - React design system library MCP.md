# Item 9: React design system library MCP

Source: [https://alexocallaghan.com/react-design-system-library-mcp](https://alexocallaghan.com/react-design-system-library-mcp)

Summary:
Mintel describes how they expose their internal React design system documentation via a local MCP (Model Context Protocol) server, using Storybook's @storybook/addon-mcp. This setup allows AI agents and developers to query component documentation programmatically, supporting automation and migration tasks. The article details the build process, CLI integration, and usage patterns for teams adopting the MCP server.

Key takeaways:
- Storybook 10 and @storybook/addon-mcp enable structured, queryable component documentation via MCP servers.
- MCP servers can be started locally or per-project, making design system docs accessible to AI agents and dev tools.
- The approach supports advanced prompts, migration planning, and automated component usage analysis.
- Includes practical build and CLI setup instructions.
