# Item 2: Navigation API - Available in all browsers soon

Source: [https://bsky.app/profile/did:plc:hxmev3uady7j4litwnr5fzbg/post/3m4uqq6kplc24](https://bsky.app/profile/did:plc:hxmev3uady7j4litwnr5fzbg/post/3m4uqq6kplc24)

Summary:
The Navigation API is set to become available across all browsers, offering a modern, centralized way to handle navigation events in client-side routers and SPAs. It aims to replace the older History API (popState, link click.preventDefault) with a cleaner, more robust interface for intercepting navigation.

Key takeaways:
- Navigation API will soon be standard in all browsers.
- Provides a unified, centralized method for handling navigation events.
- Expected to simplify and modernize SPA/router implementations.
