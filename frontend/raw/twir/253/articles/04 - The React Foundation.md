# Item 4: The React Foundation

Source: [https://react.dev/blog/2025/10/07/introducing-the-react-foundation](https://react.dev/blog/2025/10/07/introducing-the-react-foundation)

Summary:
React and React Native are moving from Meta to a new React Foundation, with a vendor-neutral governance structure and support from major industry players. The foundation will manage infrastructure, organize events, and support ecosystem projects, while technical direction will be set independently by contributors. Community feedback will be sought for the governance model, aiming to ensure React’s long-term sustainability and neutrality.

Key takeaways:
- React Foundation will oversee React, React Native, and related projects, focusing on community and ecosystem support.
- Founding members include Amazon, Callstack, Expo, Meta, Microsoft, Software Mansion, and Vercel.
- Technical governance will be independent from the foundation’s board, with community input.
- The move aims to secure React’s future and broaden its support base.
