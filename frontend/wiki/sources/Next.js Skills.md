# Next.js Skills

This source captures the emergence of agent-oriented operational knowledge packaged as reusable Next.js skills.

## Summary

- Vercel published a set of modular skills for common Next.js workflows, including file conventions, RSC boundaries, async patterns, cache components, and upgrades.
- The material treats framework knowledge as machine-usable workflow guidance rather than only human-facing documentation.
- This reinforces the broader signal that framework ergonomics increasingly include how well agents can operate inside framework constraints.

## Why This Source Matters

- It adds supporting evidence to the AI/framework branch without requiring a new top-level node.
- It also makes the `Next.js Agentic Future` page less dependent on one directional source.

## Caveats

- This is supporting tooling material, not a broad framework architecture source by itself.
