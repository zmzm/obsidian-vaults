# Item 3: PodRocket - React got hacked with David Mytton

Source: [https://podrocket.logrocket.com/react2shell-javascript-security-wake-up-call-david-mytton](https://podrocket.logrocket.com/react2shell-javascript-security-wake-up-call-david-mytton)

Summary:
This podcast episode discusses the React2Shell vulnerability, focusing on how server-side features in React 19 expanded the attack surface and the broader implications for JavaScript supply chain security. The conversation covers the role of cloud provider mitigations, dependency management challenges, and the importance of proactive security practices in modern JavaScript applications.

Key takeaways:
- React2Shell exposed serious remote code execution risks in React Server Components and Next.js.
- Server-side features can introduce new vectors for exploitation.
- Rushed patches and dependency sprawl increase overall risk.
- Security needs to be integrated early in the development lifecycle.
