# This Week in React #258

## TL;DR

- TanStack DB 0.5 introduces Query-Driven Sync, letting components declare data as queries that become precise API calls automatically
- Vercel now natively supports TanStack Start with Fluid compute and Active CPU pricing
- React 19 changes error boundary behavior: bails out after first error instead of double-rendering
- Dead Framework Theory: React has become de facto web platform, reinforced by LLMs and tooling that preferentially output React
- StyleX: Meta's styling system that compiles JS-defined styles to atomic, collision-free CSS for scale
- The Illusion of Language: React/Next.js directives are tool-level signals, not JS language features
- Error handling with react-error-boundary: flexible, composable error handling with fallbacks and async support
- Don't Blindly Use useTransition Everywhere: critiques overuse patterns and suggests alternatives
