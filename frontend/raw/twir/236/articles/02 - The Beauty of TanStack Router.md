# Item 2: The Beauty of TanStack Router

Source: [https://tkdodo.eu/blog/the-beauty-of-tan-stack-router](https://tkdodo.eu/blog/the-beauty-of-tan-stack-router)

Summary:
TanStack Router is highlighted for its type-safe, TypeScript-first approach to routing in React applications. Unlike traditional routers, it provides strict type inference for route parameters and search params, making navigation and parameter usage safer and more maintainable. The router validates search params at the route definition level and offers a developer experience that reduces runtime errors and improves codebase readability.

Key takeaways:
- Type-safe routing with full TypeScript inference for params and links.
- Route and search params can be validated and inferred, reducing manual type assertions.
- The Link component and useParams hook are type-aware, preventing invalid navigation at compile time.
- Schema validation for search params is built-in, supporting libraries like arktype.
