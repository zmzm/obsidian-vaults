# Item 9: Building an InertiaJS app with Rails

Source: [https://avohq.io/blog/inertia-js-with-rails](https://avohq.io/blog/inertia-js-with-rails)

Summary:
The article demonstrates integrating InertiaJS with Rails to build single-page applications without a separate API layer. It explains how Inertia bridges backend and frontend, allowing React components to replace traditional Rails views while preserving Rails’ strengths. Key concepts include page/layout components, data sharing via props and hooks, and enhanced navigation with the Inertia Link component.

Key takeaways:
- InertiaJS enables SPA-like experiences in Rails apps using React (or Vue/Svelte) without a dedicated API.
- Data is passed from Rails controllers to React components via props and the page object.
- Layouts and persistent layouts allow for flexible UI composition.
- The Inertia Link component provides client-side navigation without full page reloads.
