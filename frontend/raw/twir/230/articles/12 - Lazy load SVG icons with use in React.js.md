# Item 13: Lazy load SVG icons with <use/> in React.js

Source: [https://www.jeantinland.com/blog/lazy-load-svg-icons-with-use-react-js/](https://www.jeantinland.com/blog/lazy-load-svg-icons-with-use-react-js/)

Summary:
This article details how to optimize SVG icon usage in React by leveraging the `<use>` element, SVG sprites, and lazy loading via Intersection Observer. It explains how to minimize bundle size by referencing external SVGs, split sprites for granular loading, and implement a React component that loads icons only when they enter the viewport.

Key takeaways:
- Using `<use>` with SVG sprites reduces duplication and bundle size.
- Splitting sprites into individual files allows for granular, on-demand loading.
- Lazy loading with Intersection Observer ensures icons are only fetched when needed, improving performance.
- Provides a reusable React component pattern for lazy-loaded SVG icons.
