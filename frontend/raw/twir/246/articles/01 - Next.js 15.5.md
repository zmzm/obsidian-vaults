# Item 1: Next.js 15.5

Source: [https://nextjs.org/blog/next-15-5](https://nextjs.org/blog/next-15-5)

Summary:
Next.js 15.5 introduces Turbopack builds in beta, making production builds significantly faster, especially for large projects. Node.js middleware support is now stable, enabling more complex middleware logic. The release also brings TypeScript improvements, including stable typed routes and better route export validation, and deprecates the `next lint` command while warning about upcoming Next.js 16 deprecations. Turbopack is now recommended for production builds, but some edge cases and differences with Webpack remain.

Key takeaways:
- Turbopack builds are now in beta for production, offering major speed improvements, especially on large codebases.
- Node.js middleware runtime is stable, allowing full Node.js API access in middleware.
- TypeScript support is enhanced with stable typed routes and automatic type helpers for route props.
- `next lint` is deprecated; users should prepare for further deprecations in Next.js 16.
