# Item 12: Server-side React that renders as png, pdf, or interactive webapp

Source: [https://swizec.com/blog/server-side-react-that-renders-as-png-pdf-or-interactive-webapp/](https://swizec.com/blog/server-side-react-that-renders-as-png-pdf-or-interactive-webapp/)

Summary:
This article showcases a technique for rendering React components as PNG, PDF, static HTML, or a fully interactive web app, all from the same codebase, using TanStack Start. By switching the server response based on URL parameters, the app can serve different formats without code duplication. The approach leverages SSR, data preloading, and format-specific handlers to deliver flexible output for various use cases.

Key takeaways:
- React components can be rendered to multiple formats (PNG, PDF, HTML, SPA) via SSR.
- Uses TanStack Start for URL-deterministic routing and data loading.
- Preloading data ensures consistent output across formats.
- Enables new workflows (e.g., generating images or PDFs from UI components).
