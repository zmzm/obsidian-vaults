# Item 1: Announcing TanStack Start 1.0 - Release Candidate

Source: [https://tanstack.com/blog/announcing-tanstack-start-v1](https://tanstack.com/blog/announcing-tanstack-start-v1)

Summary:
TanStack Start has reached its 1.0 Release Candidate, signaling readiness for production feedback and final polish before the stable release. The framework focuses on type-safe, file-based routing, server-first ergonomics, and deep integration with TanStack Query, aiming to offer high-performance React apps without heavy abstractions. React Server Components support is in active development and will be included in a future v1.x release. Community feedback is encouraged to help finalize the stable release.

Key takeaways:
- Type-safe, file-based routing and server-first ergonomics are core features.
- Built-in streaming and planned React Server Components support.
- Deep integration with TanStack Query for data prefetching and hydration.
- No major API changes expected before 1.0; feedback is actively sought.
