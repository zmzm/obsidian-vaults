# Item 5: Functional HTML

Source: [https://overreacted.io/functional-html/](https://overreacted.io/functional-html/)

Summary:
Dan Abramov explores a reimagined HTML where custom tags are JavaScript functions, supporting attributes, objects, and async data fetching. The article walks through progressively enhancing HTML with features inspired by React’s component model, including server-side execution, JSON-based serialization, and async server tags. The piece encourages thinking about HTML and UI rendering from first principles, highlighting the conceptual underpinnings of React Server Components.

Key takeaways:
- Imagines HTML as a functional, extensible system, closely paralleling React’s model.
- Supports passing objects and async data directly into markup, serialized as JSON.
- Demonstrates how server-side rendering and componentization can be natural extensions of HTML.
- Encourages a foundational perspective on UI rendering and data flow.
