# Item 3: Next.js MCP PR - New get_routes tool

Source: [https://github.com/vercel/next.js/pull/85773](https://github.com/vercel/next.js/pull/85773)

Summary:
Next.js introduces a get_routes tool that scans the filesystem to discover all routes in a project, covering both App Router and Pages Router entry points. The tool provides an organized overview of pages and API handlers, facilitating route management and debugging in large codebases.

Key takeaways:
- get_routes helps visualize and manage all route entry points in Next.js apps.
- Supports both app/ and pages/ directory conventions.
- Useful for auditing, debugging, and documentation of project structure.
