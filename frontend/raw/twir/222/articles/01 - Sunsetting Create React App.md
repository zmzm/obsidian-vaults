# Item 1: Sunsetting Create React App

Source: [https://react.dev/blog/2025/02/14/sunsetting-create-react-app](https://react.dev/blog/2025/02/14/sunsetting-create-react-app)

Summary:
The React team has officially deprecated Create React App (CRA) for new projects, recommending developers use a modern framework (like Next.js) or build tools such as Vite, Parcel, or RSBuild instead. CRA will remain in maintenance mode and has been updated to support React 19, but new projects will see a deprecation warning. The documentation now includes migration guides and advice on when to use a framework versus a custom setup, highlighting the limitations of build tools for production apps (e.g., lack of built-in routing and data-fetching solutions). The post provides a deep dive into the rationale, migration paths, and technical trade-offs.

Key takeaways:
- CRA is deprecated for new apps; use a framework or modern build tool instead.
- Migration guides are available for Next.js, React Router, Expo, Vite, Parcel, and RSBuild.
- Frameworks are recommended for most apps, especially those needing routing and data-fetching.
- CRA and similar build tools lack production-ready features out of the box.
