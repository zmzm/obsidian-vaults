# Item 8: Styled Components - Thank you

Source: [https://opencollective.com/styled-components/updates/thank-you](https://opencollective.com/styled-components/updates/thank-you)

Summary:
This update from the core maintainer of styled-components reflects on the declining relevance of CSS-in-JS and the de facto deprecation of React’s Context API (which styled-components depends on) in React Server Components (RSC). The maintainer notes a shift in the ecosystem toward alternatives like Tailwind CSS and signals reduced active development and usage of styled-components in large apps.

Key takeaways:
- React Context API is not available in RSC, with no migration path, impacting styled-components.
- CSS-in-JS is losing popularity in favor of utility-first frameworks like Tailwind.
- The maintainer is stepping back due to lack of real-world usage and ecosystem changes.
- Projects relying on styled-components should consider migration strategies.
