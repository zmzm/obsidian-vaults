# Item 9: Driving 3D scenes in Blender with React

Source: [https://romanliutikov.com/blog/driving-3d-scenes-in-blender-with-react](https://romanliutikov.com/blog/driving-3d-scenes-in-blender-with-react)

Summary:
This article explores using a custom React reconciler to declaratively create and manage Blender 3D scenes from React code. By embedding a JavaScript engine in Blender and mapping React components to Blender objects and materials, developers can leverage familiar React patterns (including hooks) for 3D scene construction and animation.

Key takeaways:
- Custom React reconcilers can target non-DOM environments, such as Blender’s 3D API.
- Blender’s Python API is exposed to JavaScript via an embedded engine, enabling React-driven scene updates.
- Components, props, and hooks work as expected, supporting real-time updates and animation.
- Materials and scene hierarchy are managed using React’s component tree model.
