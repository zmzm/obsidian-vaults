# Item 1: nuqs 2.5 - Debounce, Standard Schema, TanStack Router, Key isolation, and more

Source: [https://nuqs.47ng.com/blog/nuqs-2.5](https://nuqs.47ng.com/blog/nuqs-2.5)

Summary:
nuqs 2.5 introduces several significant features, including debounced URL updates for search inputs, a new Standard Schema for type-safe validation, key isolation to optimize component re-renders, and experimental TanStack Router support. The release also adds global adapter defaults and early support for Next.js 15.5 typed routes. The documentation provides detailed API explanations and migration tips.

Key takeaways:
- Debounced URL updates prevent excessive network requests from high-frequency inputs.
- Standard Schema enables type-safe validation and integration with tools like tRPC and TanStack Router.
- Key isolation ensures components only re-render when relevant URL search params change, improving performance (except in Next.js, due to context limitations).
- Experimental support for TanStack Router and initial steps toward Next.js typed routes.
