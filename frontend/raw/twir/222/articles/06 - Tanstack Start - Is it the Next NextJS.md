# Item 12: Tanstack Start - Is it the Next NextJS?

Source: [https://www.robinwieruch.de/react-libraries/](https://www.robinwieruch.de/react-libraries/)

Summary:
TanStack Start is a new full-stack JavaScript framework from the creators of TanStack Query and Router, offering SSR, streaming, server functions, and deep integration with TanStack Router. It emphasizes transparency and flexibility over the "black-box" nature of frameworks like Next.js, trading some boilerplate for greater customization. Currently in beta, it's not yet production-ready but is well-documented and positioned as a framework to watch.

Key takeaways:
- TanStack Start offers SSR, streaming, server functions, and type-safe routing.
- Prioritizes transparency and flexibility over convention.
- Not production-ready yet, but documentation and examples are available.
- Designed for developers seeking more control than Next.js provides.
