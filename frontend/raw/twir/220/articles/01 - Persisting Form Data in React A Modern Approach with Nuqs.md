# Item 5: Persisting Form Data in React: A Modern Approach with Nuqs

Source: [https://armand-salle.fr/post/persisting-form-data-in-react-a-modern-approach-with-nuqs/](https://armand-salle.fr/post/persisting-form-data-in-react-a-modern-approach-with-nuqs/)

Summary:
This article introduces nuqs, a type-safe query state manager for React, as a modern alternative to localStorage for persisting form data. By storing form state in the URL, nuqs enables automatic persistence, shareability, and type safety (with Zod validation), and integrates with React Router and other major frameworks. The guide provides code samples for form setup, router integration, history navigation, and default value handling.

Key takeaways:
- nuqs stores form state in the URL, making it shareable and persistent across reloads.
- Type safety and validation are built-in via Zod.
- Integrates with React Router, Next.js, and Remix for seamless navigation/history.
- No backend required; works entirely client-side.
