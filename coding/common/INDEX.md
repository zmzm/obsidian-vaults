# Common Skills Index

Quick reference for selecting the right cross-cutting skill.

## Skill Selection Guide

| You're working on...                                   | Use skill        |
| ------------------------------------------------------ | ---------------- |
| Serena-assisted navigation and codebase workflows      | `serena`         |
| Creating, packaging, or validating new skills          | `skill-creator`  |
| General execution discipline for multi-step work       | `task-workflow`  |

## Skill Summaries

| Skill           | Purpose                                                             |
| ----------------| ------------------------------------------------------------------- |
| `serena`        | Guidance for using Serena effectively in code navigation/edit flows |
| `skill-creator` | Templates, scripts, and rules for creating new skills              |
| `task-workflow` | Cross-cutting workflow discipline for structured implementation     |

## Decision Hints

```
Is the task about the coding vault itself?
├── Creating or editing skills as artifacts? → skill-creator
├── Using Serena to inspect or modify a codebase? → serena
└── Organizing the work process across steps? → task-workflow
```
