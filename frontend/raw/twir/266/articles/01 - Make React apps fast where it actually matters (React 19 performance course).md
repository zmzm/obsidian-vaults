# Item 1: Make React apps fast where it actually matters (React 19 performance course)

Source: [https://frontendmasters.com/courses/react-performance-v2/?utm_source=reactweekly&utm_medium=email&utm_campaign=reactperf](https://frontendmasters.com/courses/react-performance-v2/?utm_source=reactweekly&utm_medium=email&utm_campaign=reactperf)

Summary:
This is a comprehensive course focused on diagnosing and fixing real-world performance bottlenecks in React apps, with a strong emphasis on React 19 features. It covers hydration, suspense, resource loading, server actions, and classic optimization techniques like memoization and code splitting. The course is structured with practical labs, code walkthroughs, and in-depth explanations of React internals such as Fiber, rendering phases, and lanes. It also demonstrates the use of React DevTools, transitions, deferred values, and hands-on exercises for state management and memoization.

Key takeaways:
- Deep dive into React 19 performance features: hydration, suspense, resource loading, server actions.
- Practical labs and code examples for identifying and resolving performance issues.
- Covers React Fiber internals, rendering phases, and lanes for prioritizing updates.
- Teaches memoization (useCallback, useMemo, React.memo), transitions, and deferred values for optimizing re-renders and responsiveness.

Recommendation: Read fully (especially if you work on performance-critical React apps or want to leverage React 19 optimizations)
