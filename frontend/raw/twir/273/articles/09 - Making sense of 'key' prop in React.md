# Item 9: Making sense of 'key' prop in React

Source: [https://inside-react.vercel.app/blog/making-sense-of-key-prop-in-react](https://inside-react.vercel.app/blog/making-sense-of-key-prop-in-react)

Summary:
The blog demystifies the purpose of the key prop in React, explaining how it helps React efficiently reconcile lists and avoid unnecessary DOM operations. Through practical examples, it clarifies why using indexes as keys is suboptimal and how keys affect component identity and state retention.

Key takeaways:
- Keys help React correctly identify and update list items during reconciliation.
- Using indexes as keys can lead to bugs and unwanted UI behavior.
- Keys are essential for lists but usually unnecessary for standalone components.
- The article includes interactive code samples illustrating key behaviors.
