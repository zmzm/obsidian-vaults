# Item 3: React Compiler oxc plugin

Source: [https://github.com/oxc-project/oxc/issues/10048](https://github.com/oxc-project/oxc/issues/10048)

Summary:
A GitHub issue discusses collaboration between the React team and the oxc project to develop a React Compiler plugin for oxc, similar to ongoing work with swc. The goal is to allow React apps to eliminate Babel pipelines by running the compiler directly in oxc, improving build performance and developer experience. The project is in early stages, with a call for contributors and collaboration.

Key takeaways:
- React Compiler support for oxc is in planning, aiming to streamline builds and reduce reliance on Babel.
- Collaboration is encouraged; early adopters and contributors are invited to participate.
- Progress with swc integration is further along, serving as a reference point.
