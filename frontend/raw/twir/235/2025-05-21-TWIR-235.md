# This Week in React #235 (MOC)

## TL;DR
- React Router has released a preview of React Server Components (RSC) support, allowing incremental adoption in existing apps and new RSC-first routes
- A new pull request introduces the skeleton for a createStore() API in React, laying groundwork for a future store-based state management solution
- Multiple pull requests show active development on <SuspenseList> in React, with improvements to child component support, reveal order, error handling, and integration with view transitions
- A new API, onDefaultTransitionIndicator, is proposed to trigger a native browser loading spinner during React transitions when no custom loading state is provided
- SWC has added an initial package, @swc/react-compiler, to support future React Compiler features
