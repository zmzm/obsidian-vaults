# Item 14: Should you use MobX-State-Tree with React in 2025?

Source: [https://coolsoftware.dev/blog/mobx-state-tree-react-2025/](https://coolsoftware.dev/blog/mobx-state-tree-react-2025/)

Summary:
MobX-State-Tree (MST) remains a viable state management solution for React, even as React 19 and the React Compiler evolve. While MST is not yet fully compatible with the React Compiler, work is underway to ensure future support. The article notes that many projects may not need third-party state management, but MST's strengths—such as portability, type safety, and functional guarantees—still make it valuable for complex applications.

Key takeaways:
- MST is still relevant and will be updated for React Compiler compatibility.
- MST offers benefits like portability and type safety not found in React's built-in state.
- Many modern React apps can avoid third-party state, but MST is useful for complex cases.
- Choice of state management is increasingly a matter of preference and project needs.
