# Item 6: Intro to performance of React Server Components

Source: [https://calendar.perfplanet.com/2025/intro-to-performance-of-react-server-components/](https://calendar.perfplanet.com/2025/intro-to-performance-of-react-server-components/)

Summary:
This article introduces the performance characteristics of React Server Components (RSC), comparing them to traditional client-side and server-side rendering approaches. It explains how RSC shifts work to the server, reducing client JavaScript and potentially improving initial load times, but also highlights the complexity of measuring and understanding these benefits due to deep integration with frameworks and bundlers.

Key takeaways:
- RSC aims to improve performance by reducing client-side JavaScript and prefetching data on the server.
- Measuring RSC performance requires a solid understanding of both client and server rendering models.
- The article provides hands-on experiments and metrics for comparing rendering strategies.
- Full technical deep dive is available in a longer companion article.
